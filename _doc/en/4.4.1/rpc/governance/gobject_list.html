--- name: gobject_list btcversion: 4.4.1 btcgroup: governance permalink: en/doc/4.4.1/rpc/governance/gobject_list/ --- gobject_list ( "signal" "type" ) List governance objects (can be filtered by signal and/or object type) Arguments: 1. signal (string, optional, default="valid") Cached signal, possible values: [valid|funding|delete|endorsed|all]. 2. type (string, optional, default="all") Object type, possible values: [proposals|triggers|all]. Examples: > syscoin-cli gobject_list > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "gobject_list", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:8370/
