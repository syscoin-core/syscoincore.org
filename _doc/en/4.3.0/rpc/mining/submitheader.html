--- name: submitheader btcversion: 4.3.0 btcgroup: mining permalink: en/doc/4.3.0/rpc/mining/submitheader/ --- submitheader "hexdata" Decode the given hexdata as a header and submit it as a candidate chain tip if valid. Throws when the header is invalid. Arguments: 1. hexdata (string, required) the hex-encoded block header data Result: null (json null) None Examples: > syscoin-cli submitheader "aabbcc" > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "submitheader", "params": ["aabbcc"]}' -H 'content-type: text/plain;' http://127.0.0.1:8370/
